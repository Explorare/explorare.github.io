Guide on Packaging and Import Baidu Exporter to Chrome

Posted on 2016-10-16   |  

It’s been a loooooong time since the last update. Just cause I’m too lazy. X P

Chrome removed Baidu Exporter from Chrome App Store due to some unknown Python transaction recently. Due to security reason Chrome will disable extensions import locally with unknown source. You will get annoying popups every time launching Chrome if you load unpacked extensions.

So in this thread I’ll give a better solution to load Baidu Exporter with the latest build using Windows Group Policy. This solution inspired by ungrown@Github, thanks.

0x01
Clone or download the latest source of Baidu Exporter from here.

0x02
Launch Chrome, open chrome://extensions, Pack extension. Browse the source you just downloaded, Pack Extension. (You don’t need to browse private key file if you do this the first time, it will generate a new one automaticly.) The newly built extension will be found in the source folder with a *.pem file, Keep Them All, you will need that .pem file if you want to update the extension in the furture. Drag the *.crx file to the chrome://extension page to load it. And you will find the extension been disabled if you relaunch the Chrome. By the way, copy the extension id to somewhere like a notepad. We will need it next.

0x03
Import the Chrome policy templates using Group Policy Editor:
a. Download the zip file of Chrome templates from here.Extract the ./Windows/adm/en-US/chrome.adm.
b. Navigate to Start > Run: gpedit.msc.
c. Navigate to Local Computer Policy > omputer Configuration > Administrative Templates.
d. Right-click Administrative Templates, and select Add/Remove Templates.
e. Add the chrome.adm template via the dialog.
f. Once complete, a Google / Google Chrome folder will appear under Administrative Templates if it’s not already there. If you added the ADM template on Windows 7 or 8.1, it will appear under Classic Administrative Templates / Google / Google Chrome.

0x04
Add the extension by the following steps:
a. Navigate to Classic Administrative Templates / Google / Google Chrome / Extensions secition.

b. Double click to open Configure extension installation whitelist on the right side.

c. Enable the policy and add the extenison id to the whitelist.

0x05
Relaunch Chrome you will find the Baidu Exporter is enabled by default now.
